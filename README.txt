WEB PORTOFOLIO MAZAYA ANDINI

Cara pakai:
1. Ekstrak file ZIP.
2. Buka index.html di browser.
3. Edit teks di index.html bila ingin mengubah data profil.
4. Edit warna/layout di style.css.
5. Edit animasi/interaksi di script.js.

Struktur:
- index.html
- style.css
- script.js
- assets/img/        = foto dan dokumentasi dari portfolio PDF
- assets/docs/       = file PDF portfolio

Fitur:
- Responsive desktop/tablet/mobile
- Animasi scroll reveal
- Scroll progress bar
- Menu mobile
- Active section navigation
- Interactive experience tabs
- Animated counters
- Tilt hover cards
- Futuristic elegant business theme
